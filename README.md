# POKE-API

Poke API its a koa js wrapper for `poke.api`.

## Installation

```bash
npm i || yarn
```

## Usage

Copy .env.example and renamed to .env, and don't forget to add required keys!

Start dev server:

```bash
npm run dev
```

Run test:

```bash
npm t
```
